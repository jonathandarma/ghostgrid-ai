# GhostGrid AI

## Mapping the Invisible Digital Workforce

GhostGrid AI is an AI-powered web intelligence platform that uncovers hidden digital labor ecosystems through workforce analytics, graph intelligence, and predictive AI forecasting.

## Built For
- Web Data UNLOCKED Hackathon by lablab.ai
- IBM BOB AI Hackathon compatible architecture

## Features
- Workforce heatmaps
- Emerging skill analysis
- Trend forecasting
- Hidden labor network graphs
- Workforce intelligence dashboard

## Technologies
- Python
- Streamlit
- Plotly
- NetworkX
- AI/ML concepts

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
