# GhostGrid AI

## Mapping the Invisible Digital Workforce

GhostGrid AI is an AI-powered workforce intelligence platform that uncovers hidden digital labor ecosystems through web data analytics, graph intelligence, and AI-driven forecasting.

## Features
- Workforce heatmaps
- Emerging skill analysis
- Trend forecasting
- Hidden labor network graph
- AI-powered workforce insights

## Technologies
- Python
- Streamlit
- Plotly
- NetworkX
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
